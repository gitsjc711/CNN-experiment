# CNN Experiment Framework

## 项目说明

这是一个基于语雀学习内容实现的轻量级卷积神经网络框架，主要用于本人深度学习实践。

### 层类内容
-  **全连接层** (Fully Connected Layer)
-  **卷积层** (Convolutional Layer) 
-  **池化层** (Pooling Layer)
